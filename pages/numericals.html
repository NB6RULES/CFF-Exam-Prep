N1.1
Air at V=250 m/s, T=5°C, p=40 kPa, D=100 mm duct. Find stagnation properties and mass flow rate. (Sept 2024 II-b)
Numerical🔥 Sept 20248 Marks
›
GIVEN: V=250 m/s, T=278K, p=40 kPa, D=0.1 m, γ=1.4, R=287 J/kg·K
Solution
  1. a = √(γRT) = √(1.4×287×278) = 334.3 m/s
  2. M = V/a = 250/334.3 = 0.748
  3. T₀ = T(1+0.2M²) = 278×(1+0.2×0.559) = 278×1.1118 = 309.1 K
  4. p₀ = p×(T₀/T)^3.5 = 40×(1.1118)^3.5 = 40×1.446 = 57.8 kPa
  5. ρ = p/RT = 40000/(287×278) = 0.5013 kg/m³
  6. A = π/4×0.01 = 7.854×10⁻³ m²
  7. ṁ = ρAV = 0.5013×7.854×10⁻³×250 = 0.984 kg/s
ANSWERS: M=0.748, T₀=309.1 K, p₀=57.8 kPa, ṁ=0.984 kg/s
N1.2
C-D nozzle: p₀=0.7 MPa, T₀=350 K, throat area=4.5 cm². Find mass flow rate and outlet area for p_exit=0.1 MPa. (Sept 2024 III-b)
Numerical🔥 Sept 20248 Marks
›
GIVEN: p₀=700 kPa, T₀=350 K, A*=4.5×10⁻⁴ m², p_exit=100 kPa, γ=1.4, R=287
Step 1 — Throat (Critical) Conditions
T* = T₀×2/(γ+1) = 350×2/2.4 = 291.7 K p* = p₀×0.5283 = 700×0.5283 = 369.8 kPa ρ* = p*/RT* = 369800/(287×291.7) = 4.417 kg/m³ a* = √(1.4×287×291.7) = 342.4 m/s
Step 2 — Mass Flow Rate
ṁ = ρ*A*V* = 4.417 × 4.5×10⁻⁴ × 342.4 = 0.681 kg/s
Step 3 — Exit Mach Number
p₀/p_exit = 700/100 = 7.0 = (1+0.2M²)^3.5 (1+0.2M²) = 7.0^(1/3.5) = 7.0^0.2857 = 1.7437 M² = (1.7437-1)/0.2 = 3.718 M_exit = 1.928
Step 4 — Exit Area from A/A* formula
A/A* = (1/M)[(1+0.2M²)/1.2]^3 at M=1.928: = (1/1.928)[(1.7437)/1.2]^3 = 0.5186 × (1.453)^3 = 0.5186×3.070 = 1.592 A_exit = 1.592 × 4.5 = 7.16 cm²
ANSWERS: ṁ = 0.681 kg/s, M_exit = 1.93, A_exit = 7.16 cm²
N1.3
Gas: Y=1.3, R=469 J/kg·K, T=593 K, M=1.2. Find velocity of sound, stagnation conditions, enthalpy, and maximum attainable velocity. (Jul 2023 II-b)
Numerical🔥 Jul 20238 Marks
›
GIVEN: γ=1.3, R=469 J/kg·K, T=593 K, M=1.2
Solution
  1. Local speed of sound: a = √(γRT) = √(1.3×469×593) = √(361,682) = 601.4 m/s
  2. Velocity: V = M×a = 1.2×601.4 = 721.7 m/s
  3. Cₚ = γR/(γ-1) = 1.3×469/0.3 = 2032 J/kg·K
  4. Stagnation temperature: T₀ = T(1+(γ-1)/2×M²) = 593×(1+0.15×1.44) = 593×1.216 = 721.1 K
  5. Stagnation enthalpy: h₀ = CₚT₀ = 2032×721.1 = 1,465,271 J/kg ≈ 1465 kJ/kg
  6. Local enthalpy: h = CₚT = 2032×593 = 1,204,976 J/kg ≈ 1205 kJ/kg
  7. Stagnation speed of sound: a₀ = √(γRT₀) = √(1.3×469×721.1) = 663.0 m/s
  8. Maximum velocity: V_max = a₀√(2/(γ-1)) = 663.0×√(2/0.3) = 663.0×2.582 = 1712 m/s
ANSWERS: a=601.4 m/s, V=721.7 m/s, T₀=721.1 K, h₀=1465 kJ/kg, V_max=1712 m/s
N1.4
Reservoir p₀=7 bar, T₀=600 K discharged through nozzle to p=1 bar. Mass flow=1 kg/s. Find throat area/pressure/velocity, exit area/Mach, and maximum velocity. (Apr 2023 III-b)
Numerical🔥 Apr 20238 Marks
›
GIVEN: p₀=700 kPa, T₀=600 K, p_exit=100 kPa, ṁ=1 kg/s, γ=1.4, R=287
Throat Conditions
T* = 600×0.8333 = 500 K p* = 700×0.5283 = 369.8 kPa ρ* = 369800/(287×500) = 2.578 kg/m³ a* = √(1.4×287×500) = 447.2 m/s = V_throat A* = ṁ/(ρ*V*) = 1/(2.578×447.2) = 8.664×10⁻⁴ m² = 8.664 cm²
Exit Mach and Area
p₀/p_exit = 7.0 = (1+0.2M²)^3.5 → M_exit = 1.928 (same as N1.2 above) A_exit/A* = 1.592 → A_exit = 1.592×8.664 = 13.79 cm² T_exit = T₀/(1+0.2×3.718) = 600/1.7437 = 344.1 K V_exit = M×a = 1.928×√(1.4×287×344.1) = 1.928×371.9 = 717.0 m/s
Maximum Velocity
V_max = √(2CₚT₀) = √(2×1005×600) = √1,206,000 = 1098.2 m/s

N2.2
Oblique shock: M₁=2, deflection δ=15°, weak shock. Find M₂, wave angle β, p₂/p₁, T₂/T₁. (Sept 2024 V-b)
Numerical🔥 Sept 20248 Marks
›
GIVEN: M₁=2, θ=15°, weak shock, γ=1.4
Solution
  1. From θ-β-M table, M₁=2, θ=15°, weak: β ≈ 45.3°
  2. M₁ₙ = M₁sinβ = 2×sin45.3° = 2×0.711 = 1.422
  3. Normal shock at M₁ₙ=1.422: M₂ₙ² = (1.422²+5)/(7×1.422²-1) = (2.022+5)/(14.155-1) = 7.022/13.155 = 0.534 → M₂ₙ=0.731
  4. p₂/p₁ = (7×1.422²-1)/6 = (14.155-1)/6 = 2.19
  5. T₂/T₁ = (7×1.422²-1)(1.422²+5)/(36×1.422²) = 13.155×7.022/72.79 = 1.269
  6. M₂ = M₂ₙ/sin(β-θ) = 0.731/sin(30.3°) = 0.731/0.505 = 1.447
ANSWERS: β=45.3°, M₂=1.447, p₂/p₁=2.19, T₂/T₁=1.269
N2.3
Normal shock: gas with γ=1.67, MW=39.9, M₁=2.5, p₁=40 kPa, T₁=253 K. Find all downstream conditions and entropy change. (Apr 2024 IV)
Numerical🔥 Apr 202412 Marks
›
GIVEN: γ=1.67 (NOT 1.4!), MW=39.9, M₁=2.5, p₁=40 kPa, T₁=253 K
Step 1 — Gas Properties
R = 8314/39.9 = 208.4 J/kg·K Cₚ = γR/(γ-1) = 1.67×208.4/0.67 = 519.4 J/kg·K
Step 2 — Shock Relations (γ=1.67)
M₁²=6.25, (γ+1)=2.67, (γ-1)=0.67 p₂/p₁ = (2×1.67×6.25-0.67)/2.67 = (20.875-0.67)/2.67 = 7.568 p₂ = 7.568×40 = 302.7 kPa T₂/T₁ = (20.205)×(2+0.67×6.25)/((2.67)²×6.25) = 20.205×6.1875/(7.1289×6.25) = 125.02/44.556 = 2.806 T₂ = 2.806×253 = 709.9 K M₂² = (0.67×6.25+2)/(20.205) = 6.1875/20.205 = 0.3062 M₂ = 0.553 ρ₂/ρ₁ = p₂T₁/(p₁T₂) = 7.568/2.806 = 2.697
Step 3 — Entropy Change
Δs = Cₚ ln(T₂/T₁) - R ln(p₂/p₁) = 519.4×ln(2.806) - 208.4×ln(7.568) = 519.4×1.032 - 208.4×2.023 = 536.0 - 421.6 = +114.4 J/kg·K ✓ (positive, as expected)
ANSWERS: p₂=302.7 kPa, T₂=710 K, M₂=0.553, Δs=+114.4 J/kg·K
📝 CRITICAL: γ=1.67 not 1.4. Derive every formula from scratch. Do NOT use air tables.
N2.4
Subsonic diffuser: area ratio 4:1, p₀=1.013 bar, T₀=290 K, M_approach=2.2. Normal shock at entry. Find exit M, T, and p₀ loss. (Apr 2023 IV-b)
Numerical🔥 Apr 20238 Marks
›
GIVEN: A_exit/A_entry=4, p₀₁=1.013 bar, T₀=290 K, M₁=2.2 (normal shock at entry)
Solution
  1. Normal shock at M₁=2.2: M₂ = √[(M₁²+5)/(7M₁²-1)] = √[(4.84+5)/(34.88-1)] = √[9.84/33.88] = √0.2904 = 0.539
  2. p₀₂/p₀₁ at M₁=2.2: from tables ≈ 0.628; p₀₂ = 0.628×1.013 = 0.636 bar
  3. T₀₂ = T₀₁ = 290 K (adiabatic shock)
  4. Isentropic diffusion from M₂=0.539: A₂/A*₂ from isentropic table at M=0.539 ≈ 1.272
  5. A_exit/A*₂ = 4×1.272 = 5.088 → from isentropic table (subsonic): M_exit ≈ 0.116
  6. T_exit = T₀₂/(1+0.2×M_exit²) = 290/(1+0.2×0.01346) = 290/1.00269 ≈ 289.2 K
  7. Stagnation pressure loss = p₀₁ - p₀₂ = 1.013-0.636 = 0.377 bar (37.2% loss)
ANSWERS: M_exit≈0.116, T_exit≈289 K, Δp₀=0.377 bar (37.2% loss)

N3.1
Fanno flow: air in 15 cm dia duct, M=0.5, p=300 kN/m², T=320 K, f=0.005, choked. Find length, entropy change, impulse function change, stagnation pressure loss. (Sept 2024 VI)
Numerical🔥 Sept 202412 Marks
›
GIVEN: D=0.15 m, M₁=0.5, p₁=300 kPa, T₁=320 K, f=0.005, choked (M₂=1)
Step 1 — Fanno Table Values at M₁=0.5
From Fanno tables at M=0.5 (γ=1.4): 4fL*/D = 1.069 (from standard Fanno table) p/p* = 2.138 T/T* = 1.143 p₀/p₀* = 1.340 At M₂=1 (choked): all ratios = 1.000
Step 2 — Duct Length to Choking
4fL_max/D = (4fL*/D) at M₁ - (4fL*/D) at M₂=1 = 1.069 - 0 = 1.069 L_max = 1.069 × D/(4f) = 1.069 × 0.15/(4×0.005) = 1.069 × 7.5 = 8.02 m
Step 3 — Exit (M=1) Conditions
p* = p₁/(p/p*) = 300/2.138 = 140.3 kPa T* = T₁/(T/T*) = 320/1.143 = 280.0 K p₀* = p₀₁/(p₀/p₀*): first find p₀₁ = 300×(1+0.2×0.25)^3.5 = 300×1.1862 = 355.9 kPa p₀* = 355.9/1.340 = 265.6 kPa
Step 4 — Change in Stagnation Pressure and Entropy
Δp₀ = p₀₁ - p₀* = 355.9 - 265.6 = 90.3 kPa (loss) Δs = -R ln(p₀*/p₀₁) = -287×ln(265.6/355.9) = -287×ln(0.7463) = -287×(-0.2926) = +84.0 J/kg·K
Step 5 — Impulse Function Change
Impulse function: F = p×A×(1+γM²) Ratio F/F*: from Fanno tables at M=0.5: F/F* = p(1+γM²) / [p*(1+γ)] = 2.138×(1+1.4×0.25) / (1×2.4) = 2.138×1.35/2.4 = 1.203 F* = p*×A×(1+γ) = 140.3×A×2.4 ΔF = F₁ - F* = (F/F*)×F* - F* = (F/F*-1)×F* F₁ = 1.203 × F* ΔF/A = F₁/A - F*/A = (1.203-1)×140.3×2.4 = 0.203×336.7 = 68.4 kPa
ANSWERS: L_max=8.02 m, Δs=84.0 J/kg·K, Δp₀=90.3 kPa, ΔF/A=68.4 kPa
N3.2
Rayleigh flow: air at V=150 m/s, T=60°C, p=0.5 MN/m². Heat added q=180 kJ/kg. Find final p, M, stagnation pressure change, entropy change. (Sept 2024 VII)
Numerical🔥 Sept 202412 Marks
›
GIVEN: V₁=150 m/s, T₁=333 K, p₁=500 kPa, q=180 kJ/kg, γ=1.4, R=287, Cₚ=1005 J/kg·K
Step 1 — Inlet Mach Number
a₁ = √(1.4×287×333) = 365.9 m/s M₁ = 150/365.9 = 0.410
Step 2 — Stagnation Temperatures
T₀₁ = 333×(1+0.2×0.168) = 333×1.0336 = 344.2 K T₀₂ = T₀₁ + q/Cₚ = 344.2 + 180000/1005 = 344.2 + 179.1 = 523.3 K
Step 3 — Find M₂ using Rayleigh T₀/T₀* ratio
From Rayleigh table at M₁=0.410: T₀₁/T₀* = 0.5764 (interpolated) T₀* = T₀₁/0.5764 = 344.2/0.5764 = 597.2 K T₀₂/T₀* = 523.3/597.2 = 0.8762 From Rayleigh table (subsonic), T₀/T₀*=0.8762 → M₂ ≈ 0.645
Step 4 — Final Pressure and Other Properties
From Rayleigh table: At M₁=0.410: p₁/p* = 2.082 At M₂=0.645: p₂/p* = 1.723 p₂ = p₁×(p₂/p*)/(p₁/p*) = 500×1.723/2.082 = 413.8 kPa At M₁=0.410: p₀₁/p₀* = 1.556 At M₂=0.645: p₀₂/p₀* = 1.149 p₀₁ = 500×(1+0.2×0.168)^3.5 = 500×1.121 = 560.5 kPa p₀* = 560.5/1.556 = 360.2 kPa p₀₂ = 360.2×1.149 = 413.9 kPa Δp₀ = 560.5 - 413.9 = 146.6 kPa (loss) Δs = Cₚ ln(T₀₂/T₀₁) - R ln(p₀₂/p₀₁) = 1005×ln(523.3/344.2) - 287×ln(413.9/560.5) = 1005×0.4185 - 287×(-0.3042) = 420.6 + 87.3 = +507.9 J/kg·K
ANSWERS: M₂≈0.645, p₂≈413.8 kPa, Δp₀≈146.6 kPa, Δs≈+508 J/kg·K
N3.3
Fanno flow: 50mm dia pipe, M₁=0.2, f=0.00375. Find distance to M=0.6 and M=1.0. (Apr 2023 VI)
Numerical🔥 Apr 202312 Marks
›
GIVEN: D=0.05 m, M₁=0.2, f=0.00375 (Fanning friction factor)
From Fanno Tables
At M=0.2: 4fL*/D = 14.697 At M=0.6: 4fL*/D = 0.491 At M=1.0: 4fL*/D = 0
Solution
Distance from M₁=0.2 to M=0.6: 4f(L₁-L₂)/D = 14.697 - 0.491 = 14.206 L = 14.206 × D/(4f) = 14.206 × 0.05/(4×0.00375) = 14.206 × 3.333 = 47.35 m Distance from M₁=0.2 to M=1.0 (choking): 4fL_max/D = 14.697 - 0 = 14.697 L_max = 14.697 × 0.05/(0.015) = 14.697 × 3.333 = 48.99 m
ANSWERS: L(to M=0.6) = 47.35 m, L(to M=1.0) = 48.99 m

Numerical🔥 Apr 20233 Marks
›

Solution
