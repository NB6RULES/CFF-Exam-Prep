Module 2 — Normal & Oblique Shocks
R2
Normal Shock + Oblique Shock Formulas
›
=== NORMAL SHOCK (air, γ=1.4) === M₂² = (M₁²+5)/(7M₁²-1) p₂/p₁ = (7M₁²-1)/6 T₂/T₁ = (7M₁²-1)(M₁²+5)/(36M₁²) ρ₂/ρ₁ = 6M₁²/(M₁²+5) max=6 as M→∞ T₀₂ = T₀₁ (conserved — adiabatic) p₀₂/p₀₁ < 1 (always a loss) V₁V₂ = a*² (Prandtl-Meyer) Δs = -R ln(p₀₂/p₀₁) > 0 always === GENERAL γ === M₂² = [(γ-1)M₁²+2]/[2γM₁²-(γ-1)] p₂/p₁ = [2γM₁²-(γ-1)]/(γ+1) T₂/T₁ = [2γM₁²-(γ-1)][2+(γ-1)M₁²]/[(γ+1)²M₁²] ρ₂/ρ₁ = (γ+1)M₁²/[2+(γ-1)M₁²] === OBLIQUE SHOCK === M₁ₙ = M₁ sin β (apply normal shock with M₁ₙ) M₂ = M₂ₙ/sin(β-θ) θ-β-M: tan θ = 2cotβ [M₁²sin²β-1]/[M₁²(γ+cos2β)+2] V₁ₙV₂ₙ = a*² (Prandtl, normal components) === EXPANSION WAVES === ν(M) = √((γ+1)/(γ-1)) arctan√((γ-1)/(γ+1)(M²-1)) - arctan√(M²-1) θ = ν(M₂) - ν(M₁) (isentropic, p₀ preserved)
Module 3 — Fanno & Rayleigh
R3
Fanno Flow + Rayleigh Flow Formulas
›
=== FANNO FLOW (adiabatic friction, constant area) === 4fL*/D = (1-M²)/(γM²) + (γ+1)/(2γ) ln[(γ+1)M²/(2+(γ-1)M²)] T/T* = (γ+1)/[2(1+(γ-1)/2 M²)] p/p* = (1/M)√[(γ+1)/(2+(γ-1)M²)] p₀/p₀* = (1/M)[(1+0.2M²)/1.2]^3 (air) T₀ conserved. p₀ decreases. Both subsonic and supersonic → M=1. === RAYLEIGH FLOW (frictionless, heat transfer, constant area) === T/T* = 4(γ+1)²M²/(1+γM²)² p/p* = (γ+1)/(1+γM²) T₀/T₀* = 2(γ+1)M²[2+(γ-1)M²]/(1+γM²)² p₀/p₀* = [(γ+1)/(1+γM²)][(2+(γ-1)M²)/(γ+1)]^(γ/(γ-1)) q = Cₚ(T₀₂-T₀₁) = Cₚ T₀* [(T₀₂/T₀*)-(T₀₁/T₀*)] Maximum T occurs at M=1/√γ = 0.845 (air). Cannot cross M=1 by heat addition alone in constant area. Rayleigh line in p-v: p = const - G²v (straight line, negative slope) === KEY COMPARISONS === Fanno: adiabatic, friction, T₀ constant Rayleigh: frictionless, heat added, T₀ changes Both: M→1, p₀ decreases, choking at M=1

Exam Strategy
ES
How to write — 3 mark, 4 mark, 8–12 mark answers
›
3-Mark Questions
  • 3 paragraphs/bullet points — one key idea each
  • One formula (derivation NOT needed unless asked)
  • One sketch with labels where applicable
  • ~half a page max
4-Mark Derivation Questions
  • State starting equations clearly
  • Number your steps
  • Box the final result
  • State physical meaning of result
8–12 Mark Numericals
  • GIVEN: list all data at top
  • Write formula BEFORE substituting numbers
  • Number each step
  • Box intermediate answers
  • SUMMARY at end: list all final answers
  • Check units throughout
Sketch Questions (Module 4)
  • Large, clear sketch — not tiny
  • Label ALL components with arrows
  • Schlieren: label knife edge explicitly
  • Pitot: label inner tube (total) and outer tube (static) separately
  • Wind tunnel: label settling chamber, nozzle, test section, diffuser, drive
📝 GAS TABLES ARE PERMITTED IN THE EXAM. Use them for: A/A* values, Fanno 4fL*/D, normal shock M₂, Rayleigh T₀/T₀*, isentropic p₀/p at given M. Saves enormous time on numericals.
💡 COMMON MISTAKES: (1) Forgetting to convert T to Kelvin. (2) Using γ=1.4 when problem gives different gas. (3) Using isentropic pitot formula in supersonic flow (must use Rayleigh). (4) Not stating physical meaning after derivations.
